## About FEEN

FEEN (Field Expression Encoding Notation) is a minimalistic, rule-agnostic textual format for capturing static board positions in a canonical, compact, and game-neutral way. For more information, see the [FEEN Specification](https://sashite.dev/specs/feen/1.0.0/).
